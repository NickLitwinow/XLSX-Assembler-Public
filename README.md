Пример работы программы. Данные подставлены из тестового файла и не являются показательными, но подходят для демонстрации работы программы.

### Задачи, которые выполнял, и какие технологии применял:
- Разработал ETL-процесс для автоматического объединения данных из множества Excel файлов в один с использованием Python.
- Использовал библиотеки Pandas и Openpyxl для чтения, обработки и записи данных, включая стилизацию и копирование листов Excel.
- Реализовал графический интерфейс на PyQt5, обеспечив удобный выбор файлов и отслеживание прогресса процесса объединения данных.
- Оптимизировал код и процесс обработки данных, увеличив производительность и снизив нагрузку на систему.
### Достижения:
- Сократил время обработки данных с нескольких дней до 10 минут.
- Автоматизировал процесс, ранее выполнявшийся вручную с использованием макросов в Excel, что снизило трудозатраты отдела.
- Обеспечил удобный и интуитивно понятный интерфейс, что улучшило взаимодействие пользователей с системой.
